cli Code-less approach

  1. import your Picon font 
    @font-face {
  font-family:picon;
  src: url(/url/to/picon.woff2);
}
  2. Bind it to an element/class, for example:
    rb { font-family:picon; }
  3. That's it ! You can now render the app icon app using 
    <rb>app</rb>
