# SVG Studio

SVG editor for quick icons and logos prototyping

![](https://i.imgur.com/mWN9YL4.png)

# Usage

- arrows: move focused node
- delete: delete focused node
- enter: duplicate focused node
- shift+[one of above] = apply to every node
